# 项目简介

## 核心功能
- 销售数据可视化分析
- 商品分类
- 加入购物车
- 结算商品
- 订单查询
- 个人中心
- 商品详情

## 适用场景
- 超市管理
- 在线零售
- 数据分析

## 技术特点
- 使用Python和Django框架构建
- 集成echarts进行数据可视化
- 基于MySQL数据库存储数据
